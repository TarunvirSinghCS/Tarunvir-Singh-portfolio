# Troubleshooting Case Studies

This folder contains case studies of technical problems I encountered and resolved. Each artifact demonstrates problem-solving, investigation steps, solutions, and lessons learned. These examples highlight skills in networking, virtualization, software debugging, and system administration.

## Artifacts


1. **[Python Bug Debugging](python-bug-debug.md)**  
   Shows a software debugging scenario where a Python script crashed due to a TypeError, including investigation steps and the fix.

2. **[VM Network Issue](vm-network-issue.md)**  
   Details a virtual machine unable to access the internet due to misconfigured network settings, with investigation steps and solution.
   
3. **[Wi-Fi Troubleshooting](wifi-troubleshooting.md)**  
   Explains a Wi-Fi connectivity issue, how it was diagnosed, and the solution, including tools like `ping` and `ipconfig/ifconfig`.

4. **[Windows Login Problem](windows-login-problem.md)**  
   Covers a Windows account login failure caused by cached credentials, with investigation, solution, and lessons learned.



## Purpose

These artifacts demonstrate my ability to troubleshoot real technical problems, think logically under pressure, and document the process clearly. They are written to showcase practical technical skills that employers and reviewers look for.
