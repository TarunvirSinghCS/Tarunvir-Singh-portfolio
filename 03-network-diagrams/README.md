# Network Diagrams

This folder contains a high-level network diagram illustrating how a user device connects to a web server over the internet. This artifact demonstrates understanding of network flow, security considerations, potential failures, and monitoring points.

## Artifact

**[basic-network-flow-simple.svg](basic-network-flow-simple.svg)**  
A high-level overview of the network flow:  
User Device → Router → ISP → Internet → Web Server → Response.  

**Push Further / Notes:**  
- **Security Controls:** Assumes TLS/HTTPS for encrypted communication and a basic firewall at the router.  
- **Potential Failures:** Network outage at ISP or server, router misconfiguration.  
- **Monitoring / Logging:** Network activity can be logged at the router or server; alerts could detect failures.

## Purpose

This diagram illustrates the flow of data across a network and highlights security, reliability, and monitoring considerations. It demonstrates the ability to **visually communicate technical concepts**, a key skill in IT and cybersecurity roles.
