# System Explanations

This folder contains explanations of core technical processes in computer systems. Each document is designed to show clear, step-by-step understanding, operational considerations, and potential security or failure points.

## Artifacts

1. **[Login Process](login-process.md)**  
   Explains what happens when a user logs into a computer, including authentication steps, session creation, and security considerations.

2. **[Website Request Process](website-request-process.md)**  
   Details what happens when a user types a website into a browser, including DNS lookup, HTTP requests, rendering, and security considerations.

3. **[Wi-Fi Connection Process](wifi-connection-process.md)**  
   Describes how a device connects to a Wi-Fi network, including scanning, authentication, IP assignment, and potential security threats.

## Purpose

These explanations demonstrate a foundational understanding of computer systems and networking processes. They are written in a way that shows clarity, logical flow, and awareness of operational and security aspects.
