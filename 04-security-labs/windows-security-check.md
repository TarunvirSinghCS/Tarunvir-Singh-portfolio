# Security Lab: Firewall and Endpoint Protection Review

## Objective
Review the local machine's built-in security protections and understand how they defend against threats.

## Tools Used
- Windows Security
- Windows Defender Firewall
- Update History
- Command Prompt

## Procedure
1. Checked firewall status using Windows Security.
2. Verified antivirus protection was enabled.
3. Reviewed recent scan history.
4. Checked update patch level.
5. Tested local network connectivity using ping.

## Security Importance
The firewall filters suspicious network traffic.
Antivirus helps detect malware and malicious files.
Regular updates patch known vulnerabilities.

## Reflection
This lab helped me understand how endpoint protection works at the operating system level.
In a real IT role, these checks would be part of routine device troubleshooting and security validation.
