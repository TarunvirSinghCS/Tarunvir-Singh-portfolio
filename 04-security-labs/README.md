# Security Labs

## Overview
This folder contains my Week 4 artifact focused on system security and hardening. The goal of this assignment is to understand how to reduce risks by properly configuring a system.

## Artifact
**System Hardening Checklist — Personal Laptop (Windows)**

This checklist outlines important steps to secure a personal laptop, including:
- Enabling system updates  
- Configuring firewall settings  
- Using strong authentication  
- Enabling antivirus protection  
- Disabling unnecessary services  
- Encrypting data  
- Setting up backups  

## Purpose
The purpose of this artifact is to demonstrate how security is achieved through practical configuration decisions rather than just tools. It shows how to reduce vulnerabilities and protect against common threats.

## Key Takeaway
Security is not automatic—systems must be actively configured and maintained to stay protected.
