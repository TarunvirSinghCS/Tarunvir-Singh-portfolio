# System Hardening Checklist — Personal Laptop (Windows)

## System Description
This checklist is for securing a personal Windows laptop used for school, browsing, and development work. The goal is to reduce security risks such as malware, unauthorized access, and data loss.

## Checklist

### 1. System Updates
- Enable automatic Windows updates  
- Regularly update all installed software (browser, IDEs, apps)  
- Remove outdated or unsupported programs  

### 2. Firewall Configuration
- Ensure Windows Firewall is enabled  
- Block unnecessary inbound connections  
- Review firewall rules periodically  

### 3. Strong Authentication
- Use a strong password (at least 12 characters, mix of types)  
- Enable Windows Hello (fingerprint or face recognition if available)  
- Enable multi-factor authentication (MFA) on important accounts  

### 4. Antivirus & Malware Protection
- Enable built-in antivirus (Windows Security / Defender)  
- Run regular system scans  
- Avoid downloading files from untrusted sources  

### 5. Disable Unused Services
- Turn off Bluetooth when not in use  
- Disable Remote Desktop if not needed  
- Remove unused startup programs  

### 6. Disk Encryption
- Enable BitLocker (full disk encryption)  
- Store recovery key in a secure location  

### 7. Backup Configuration
- Enable automatic backups (OneDrive or external drive)  
- Perform regular manual backups of important files  
- Test backup recovery occasionally  

### 8. Secure Network Usage
- Avoid public Wi-Fi or use a VPN  
- Connect only to trusted networks  
- Change default router passwords at home  

### 9. User Account Control
- Use a standard user account for daily tasks  
- Only use administrator privileges when necessary  

### 10. Browser Security
- Keep browser updated  
- Enable pop-up blocker  
- Install only trusted extensions  

## What Risks Still Remain?
- Phishing attacks (user may click malicious links)  
- Human error (downloading unsafe files)  
- Zero-day vulnerabilities (unknown software flaws)  
- Weak passwords on external accounts  
- Misconfigured settings over time
